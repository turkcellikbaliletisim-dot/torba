# RADIUS — Köşe Yuvarlama (Border Radius) Standartları

> **Yaklaşım:** Yumuşatılmış, dost canlısı ve modern köşe yarıçapları.

---

## 1. Border Radius Skalası

| Token | Değer (px) | Kullanım Alanı |
|---|---|---|
| `radius-sm` | 8px | Küçük etiketler, rozetler |
| `radius-md` | 12px | Form girdileri (inputlar), küçük kartlar |
| `radius-lg` | 16px | Standart kartlar, dropdown menüler |
| `radius-xl` | 20px | Büyük kampanya kartları, hero alanları |
| `radius-2xl`| 24px | Modallar, Bottom Sheet üst köşeleri |
| `radius-full`| 9999px | Yuvarlak ikon butonları, avatar, pill butonlar |
